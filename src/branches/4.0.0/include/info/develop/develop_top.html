開発用設定

デバッグモード

デバッグモード

  • 一部のセキュリティチェックがスキップされます。
  • ゲーム一覧にゲーム削除のリンクが出現します (緊急時以外はクリックしないで下さい)。

互換性

php database sound

php

  • Ver. 4.0 以降は PHP5.6 環境が必要です。
  • Ver. 1.5 以降は PHP5 環境が必要です。
  • PHP4 環境のサポートは Ver. 1.4 までとなっています。

database

  • Ver. 2.0 以降はテーブル構成が Ver. 1 と違うのでログの互換性がありません。
  • ユーザアイコンのみ互換性が保たれているので Ver. 1 からインポート可能です。

sound

  • Ver. 4.0 以降は音源フォーマットが swf から mp3 に変更されます。

改定案件

  • 過去ログ HTML 化の自動化
