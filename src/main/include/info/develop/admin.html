トップページの編集方法について

Information

  • top/information.html を編集してください。

メニュー

  • top/menu.html を編集してください。

交流用サイト

  • config/server/menu_config.php 内の $list を編集してください。

外部リンク

  • config/server/menu_config.php 内の $add_list を編集してください。

関連サーバー

  • config/server/top_page_config.php を編集してください。

掲示板情報

  • config/server/bbs_config.php を編集してください。
  • php.ini の allow_url_fopen を on にしてください。

css について

  • css/ ディレクトリ内に集約されているので必要に応じて編集してください。
  • css/base.css に記述すると全てのページに適用されます。
  • css/header.css に h1～h5 タグの共通デザインが設定されています。
  • css/talk_voice.css に基本的な発言のデザインが設定されています。
  • css/role.css に役職の色情報が設定されています。

設定について

  • config/ ディレクトリ内に集約されているので必要に応じて編集してください。
  • 設置に必要な情報は README も参照してください。
  • 特徴と仕様やルールで一部の管理者設定を参照することができます ([ 設定値 ] の形式になっている部分です) 。

関連サーバ村情報について

  • config/server/shared_server_config.php を編集してください。
  • 交流のある人狼式システムサーバの村稼働状況を取得して表示します。

サーバ更新履歴について

  • include/info/history/server.html を編集してください。
  • サーバのバージョンアップや設定変更の履歴を記録する為に使用することを想定しています。

TOPページ更新履歴について

  • include/info/history/top.html を編集してください。
  • トップページの Information の履歴を記録する為に使用することを想定しています。
