キューピッド系 天使系

基本ルール

概要 勝利条件 判定

概要

  1. 初日の夜に「恋人」を作り、生存を目指します。

勝利条件

  1. ゲーム終了時に恋人が二人以上生存している。
  2. 人狼を全滅させるか、生存カウントの村人の生存数を人狼の生存数以下にする。
  3. 勝利条件を満たした吸血鬼が存在する場合は恋人陣営勝利となる。
  4. 生存者が全て恋人になった場合は恋人陣営勝利となる。
  5. 自身や自分の作った恋人の生死は不問。

投票の仕様

  1. 初日の夜に「恋人」にする人を二人選ぶ (人数は例外あり)。
  2. 投票結果は即座に反映される。
  3. 自分以外を恋人の対象に選ぶことができる (他人撃ち) 人数の制限は管理者が設定する。

自分撃ち固定のキューピッド。
処刑投票先を洗脳者にすることができる。
人狼に襲撃された時に、洗脳者を身代わりにして生き延びる事ができる。

処刑投票洗脳能力

  1. 判定は処刑者決定後で、自分が毒やショック死で死亡した場合でも有効。
  2. 対象が死亡していた場合は無効 (例：処刑・毒死)。
  3. 自分が処刑された場合は無効。
  4. 吸血鬼系、吸血鬼をコピーした覚醒者・夢語部には無効。

身代わり能力

  1. 身代わりが発生した場合、人狼の襲撃は失敗扱い。
  2. 代わりに死んだ人の死因は「誰かの犠牲となって死亡したようです」。
  3. 本人は身代わりが発生しても分からない。
  4. 逃亡失敗・人狼に遭遇して死亡した逃亡者系を身代わりにすることはできない。
  5. 身代わり君か、襲撃者が天狼 (完全覚醒状態) だった場合は無効。

  5. 文車妖妃が複数出現してもカップルが重なっていなければ共に有効。
    例5-1) X[文車妖妃] -> A-B / Y[文車妖妃] -> C-D → X・Y 共に有効
